# f29pdtest

## Frontend Setup

1. **Navigate to the Frontend Directory**:
   ```bash
   cd frontend/pdtest/src
   ```

2. **Install Frontend Dependencies**:
   ```bash
   npm install formik react-phone-input-2 react-select react-router-dom axios
   ```

3. **Run the Frontend Application**:
   ```bash
   npm start
   ```
